Execute the following project structure and logic based on the "Gen_AI Project Documentation .pdf". The system must implement a hybrid approach combining Digital Image Processing (DIP) and Latent Diffusion to restore corrupted MNIST digits for improved OCR accuracy.  1. 2. Component SpecificationsA. Distortion Engine (data/distortion_engine.py)Requirement: Generate synthetic corruption pairs.  Functionality: Implement a class that takes a clean MNIST tensor and applies:Gaussian Noise: $\sigma \in [0.1, 0.5]$.Motion Blur: Kernel size $3\times3$ or $5\times5$.Spatial Masking: Random $8\times8$ pixel occlusion.B. DIP Layer (src/preprocessing/)Requirement: Non-adaptive stabilization.  Logic: Before entering the VAE, images must pass through a Median Filter (to remove salt-and-pepper noise) and Global Histogram Equalization (to normalize contrast).  C. Generative Backbone (src/models/vae.py)Requirement: Latent representation learning.  Architecture:Encoder: 3 Convolutional layers (stride 2) mapping to $\mu$ and $\log\sigma$.  Latent Space: $d=64$.Loss: $L = \text{ReconstructionLoss} + \beta \cdot \text{KL-Divergence}$.  D. Conditional Diffusion (src/core/diffusion_engine.py)Requirement: Stepwise denoising in latent space.  Algorithm: Latent Diffusion Model (LDM).Conditioning: Use a 1-hot encoded vector representing the predicted "Corruption Type" from a small classifier to guide the U-Net noise prediction.  E. Evaluation Logic (src/utils/metrics.py)Primary Metric: Compare OCR Accuracy $A_{clean}$ vs. $A_{corrupted}$ vs. $A_{restored}$.  3. Implementation Priorities (Execution Order)Phase 1: Script distortion_engine.py and ocr_classifier.py to establish the baseline performance drop.  Phase 2: Implement and train vae.py on clean MNIST to ensure high-fidelity reconstruction.  Phase 3: Train the unet.py on the noise-prediction task using the latent vectors generated by the frozen VAE.  Phase 4: Integrate the DIP filters into main.py as the final entry-point.  4. Technical ConstraintsInput Size: $1\times28\times28$ grayscale.  Framework: PyTorch.Optimization: Use Adam optimizer with a learning rate of $1e-4$.
